## 📌 Overview

Zentask is a professional task management system built with Laravel. It helps individuals and teams organize, track, and complete tasks efficiently with features like task management, calendar view, analytics dashboard, and user authentication.

## ✨ Features

### Core Features
- ✅ **User Authentication** - Register, Login, Logout, Profile management
- ✅ **Task Management** - Create, Read, Update, Delete tasks
- ✅ **Priority Levels** - High, Medium, Low with color-coded badges
- ✅ **Status Tracking** - Pending, In Progress, Completed
- ✅ **Calendar View** - Visualize tasks by due date
- ✅ **Analytics Dashboard** - Charts and statistics for task performance
- ✅ **Soft Delete & Restore** - Archive tasks and restore when needed
- ✅ **Responsive Design** - Works on desktop, tablet, and mobile

### Additional Features
- 🔔 **Due Date Reminders** - Visual indicators for overdue and upcoming tasks
- 📊 **Task Distribution Chart** - Doughnut chart showing task breakdown
- 📈 **Completion Rate** - Track your productivity progress
- 🎨 **Modern UI** - Clean, professional interface with custom sidebar
